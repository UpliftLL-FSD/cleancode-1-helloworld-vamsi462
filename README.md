# Assignment : Hello World (Multi-Language)

Welcome to your first assignment! The goal of this task is to ensure your local development environment is set up correctly and you know how to push code to GitHub.

You are required to write a program that prints exactly `Hello, World!` in three different programming languages: **Java**, **Python**, and **JavaScript**.

---

## 🛠️ Prerequisites

Before starting, ensure you have the following installed on your computer:
* **VSCode**
* **Java Development Kit (JDK)** (Version 11 or higher)
* **Python 3**
* **Node.js** (for running JavaScript outside the browser)
* **Git** ---

## 🚀 Tasks

### Task 1: Python
1. Navigate to the `python/` directory.
2. Open `helloworld.py`.
3. Write the code to print `Hello, World!` to the console.
4. Run and test your code locally using the terminal:
   
   python python/helloworld.py

### Task 2: JavaScript
1. Navigate to the javascript/ directory
2. Open helloworld.js
3. Write the code to print 'Hello, World!' to the console.
4. Run and test your code locally using the terminal:
   
   node javascript/helloworld.js

### Task 3: Java
1. Navigate to the java/ directory
2. Open HelloWorld.java
3. Inside the main mehtod, wrtie the code to print Hello, World! to the console.
4. Compile and run your code locally using the terminal:

   javac java/HelloWorld.java
   java -cp java HelloWorld

## How to Submit

1. Stage your changes:

   git add .

2. Commit your changes

   git commit -m "Completed Hello World in Python, JS, and Java"
   
3. Push to GitHUb

   git push origin main
